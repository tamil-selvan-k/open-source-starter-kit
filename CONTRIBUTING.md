# 🤝 Contributing Guidelines

Thank you for deciding to contribute to the **Open Source Starter Kit**! We are thrilled to have you here.

This document outlines the super simple rules for contributing to keep things tidy and fun.

## 🏆 Contribution Levels Reminder

Please ensure your contribution falls into one of these three levels:

*   **🟢 Level 1 (The First Step):** Adding your name to `contributors.md`.
*   **🟡 Level 2 (The Fixer):** Fixing typos or grammar issues **strictly** within `docs/fix-me.md`.
*   **🔵 Level 3 (The Builder):** Adding a sample project to the `projects/` directory.

## 📋 General Rules

1.  **Be Respectful:** We follow a beginner-friendly code of conduct. Be kind and encouraging to everyone.
2.  **Keep it Simple:** This repo is for learning. Please don't submit massive, complex code rewrites.
3.  **One Change per PR:** If you want to add your name AND fix a typo, please create two separate Pull Requests.
4.  **Use the PR Template:** When opening a Pull Request, please fill out the checklist provided in the template.
5.  **Level 2 Specific Rules:** Level 2 contributions are ONLY for `docs/fix-me.md`. Do not duplicate fixes, and only submit **one fix per PR**. All other files are intentionally kept clean.

## 🛠️ Step-by-Step Guide

If you need help making a Pull Request, please refer to the visual guide in our main [README.md](README.md).

We can't wait to see your Pull Request! 🚀
